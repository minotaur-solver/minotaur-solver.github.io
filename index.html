Minotaur is an open source software for solving mixed-integer nonlinear optimization problems. It features solvers for both convex and nonconvex problems. It can be extended or customized for specific problem types.

Features

  • Native support for nonlinear functions including automatic differentiation
  • Shared-memory parallel implementation
  • Threadsafe callable library
  • Compatible with AMPL and Pyomo

Solvers Available

  • bnb: nonlinear optimization based branch-and-bound for convex MINLPs
  • qg: LP/NLP based branch-and-cut for convex MINLP
  • qpd: QPDiving based branch-and-bound for convex MINLP
  • glob: Global optimization solver for quadratically constrained problems (experimental).

Download
